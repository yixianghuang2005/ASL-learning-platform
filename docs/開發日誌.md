## 2026-05-23：Kaggle 250 詞 GRU 模型訓練

### 資料前處理
- `prepare_kaggle_dataset.py`：parquet → 225 維 npz（94,477 筆）
- 發現 1.64 億個 NaN → `np.nan_to_num(X, nan=0.0)` 清除

### 第一次 Transformer 嘗試（失敗）
- DirectML 不支援 TransformerEncoderLayer 反向傳播
- loss 卡在 ln(250)=5.52（等於亂猜），放棄

### GRU 訓練成功
- `train_kaggle_gru.py`：`GRU(225→128, 2層) → Linear(64) → ReLU → Linear(250)`
- **結果：Test acc 62.2%（250 詞）**
- ONNX 匯出，複製到 `frontend/public/models/`

## 2026-05-24：改良 GRU → 準確率 69.2%

### train_kaggle_best.py
- 架構升級：`GRU(hidden=256, 3層) + LayerNorm + 混合池化（last+avg）+ GELU head`
- WarmupCosineScheduler + 資料增強升級（時間 dropout + 特徵 dropout）
- 早停於 Epoch 62
- **Val acc 70.2%，Test acc 69.2%（前版 62.2%，+7%）**
- ONNX 4.7MB，已部署前端

## 目前技術架構

### 前端推論流程

```
字母辨識：攝影機 → MediaPipe Hands (21點) → 正規化 (63維) → MLP ONNX → 字母
詞彙辨識：攝影機 → MediaPipe Holistic (75點) → 正規化 (225維) → GRU ONNX → 詞彙
動態字母：JZMotionDetector (純規則式軌跡偵測) → J/Z
```
